
# **画面遷移図**

```
[ログイン画面]
  - ログイン成功 → [マイページ]
  - 未認証状態で認証が必要なページへアクセス → [ログイン画面]

[マイページ]
  - 新規診断開始ボタン → [診断開始画面]
  - 診断履歴の再開ボタン → [診断進行画面]
  - 診断結果リンク → [診断結果画面]
  - 通知アイコン → [通知一覧画面]
  - プロフィール編集ボタン → [プロフィール管理画面]
  - グループ管理ボタン → [グループ管理画面]

[診断開始画面]
  - 質問に回答 → [診断進行画面]
  - 戻るボタン → [マイページ]

[診断進行画面]
  - 次の質問に回答 → [診断進行画面]（繰り返し）
  - 診断完了 → [診断結果画面]
  - 保存ボタン → [マイページ]

[診断結果画面]
  - 診断詳細リンク → [診断結果画面]
  - 戻るボタン → [マイページ]

[通知一覧画面]
  - 通知クリック → 該当画面（例: グループ詳細画面、診断進行画面など）
  - 戻るボタン → [マイページ]

[グループ管理画面]
  - グループ作成ボタン → [グループ詳細画面（新規作成）]
  - グループ詳細リンク → [グループ詳細画面]
  - 戻るボタン → [マイページ]

[グループ詳細画面]
  - 相性診断リンク → [相性診断結果画面]
  - メンバー追加ボタン → [グループ詳細画面（更新）]
  - 戻るボタン → [グループ管理画面]

共通動作:
  - ログアウトリンク（全画面共通） → [ログイン画面]
```
